## How to use boostrap in React js with two ways
1. install bootstrap with npm
2. use boostrap with direct cdn
3. React Boostrap (react Component Base)

---

### 1. Method one
```javascript
npm install bootstrap
```

#### Add src/main.jsx
```javascript
import 'bootstrap/dist/css/bootstrap.min.css'
import 'bootstrap/dist/js/bootstrap.bundle.min.js' 
```
* And you can use boostrap 

---

### 2. Method two
* goto bootstrap official website and copy cdn and paste index.html file


---

### 3. Method three (React Component Base)
```javascript
npm install react-bootstrap
```
**You Can use it**
```javascript
<Container>
        <Row>
          <p>Hello World</p>
          <h1>Vite + React</h1>
        </Row>
      </Container>
```
